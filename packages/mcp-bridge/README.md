# mcp-bridge

**Mission:** Making Aether "Agent-Native" via the Model Context Protocol.

* **Tech Stack:** TypeScript, MCP SDK.
* **Function:** Exposes Aether’s data as "Tools" and "Resources" for Claude, GPT, and custom internal agents.

> This package turns Aether into a standard "plugin" for the world's most powerful LLMs. It implements the 2026 Model Context Protocol, allowing any autonomous agent to safely explore enterprise data with zero custom integration.
